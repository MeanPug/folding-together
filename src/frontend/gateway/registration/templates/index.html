Help find a cure to Covid-19

Researchers need your computing power to run simulations in search of cures to diseases like Covid-19. The Folding@Home project was created to make contributing unused computing power towards this cause as simple as possible, however many are hesitant about running the required software on their personal computers.

Making contribution of computation resources easy

Folding@Together makes it easy for donors to contribute computing power to researchers via cash microdonations. These donations are deployed to launch and run powerful servers on the AWS Cloud which researchers can use for running computationally expensive protein folding simulations. Learn more about how we make contributing resources simple. Let's help find a cure, together.

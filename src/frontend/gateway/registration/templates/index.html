Help us fight against Covid-19 with Folding@Home

Though it isn't new, the Folding@Home project allows individuals to donate their computing resources to researchers searching for cures to diseases like Covid-19.

These resources allow researchers to run computationally expensive simulations that illustrate how atoms in a protein move relative to each other, allowing for therapeutic discoveries. Let's help find a cure, together.

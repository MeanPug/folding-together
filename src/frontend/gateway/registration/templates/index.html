Help find a cure to Covid-19

Researchers need your computing power to run simulations in search of cures to diseases like Covid-19. The Folding@Home project was started to make contributing unused computing power for this cause as simple as possible, however many are hesitant about running software like this on their personal computers.

Making contribution of compute resources easy

Folding@Together makes it easy for users to contribute computing power to researchers by leveraging the cloud to run powerful servers on which researchers can launch computationally expensive simulations. Learn more about how we make contributing resources simple. Let's help find a cure, together.
